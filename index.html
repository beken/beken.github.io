Recent Publications

A Multivocal Review of MLOps Practices, Challenges and Open Issues

B. Eken, S. Pallewatta, N. Tran, A. Tosun, M. A. Babar. ACM Computing Surveys, 2025.

Provides a comprehensive multivocal synthesis of 198 academic and grey sources to unify fragmented MLOps knowledge, outlining its key practices, challenges, and solutions.

Tratto: A Neuro-Symbolic Approach to Deriving Axiomatic Test Oracles

D. Molinelli, A. Martin-Lopez, E. Zackrone, B. Eken, M. D. Ernst, M. Pezzè. Proceedings of the ACM on Software Engineering (ISSTA), 2025.

Introduces a neuro-symbolic framework for automatically deriving formal test oracles from natural language requirements.

Classification and Analysis of Employee Feedback with Deep Learning Algorithms

G. Yiğidefe, S. Ç. Kaman, B. Eken. Sakarya University Journal of Computer and Information Sciences, 2025.

Presents a deep learning-based approach for classifying and interpreting employee feedback to improve organizational insights.

Leveraging Semantic Coupling Through Class Name Embeddings for Software Change Prediction

B. Eken, N. B. Oğur. International Conference on Artificial Intelligence, Metaverse and Cybersecurity, 2024.

Proposes a semantic coupling metric using class name embeddings to enhance change prediction in software systems.

Projects

Automated Short Answer Grading with LLMs

Developing a large language model–driven evaluation pipeline to automatically grade open-ended student answers in software engineering and computer science courses, focusing on prompt strategies, reasoning quality, and rubric alignment.

PI: Beyza Eken - Funded by TÜBİTAK 1002 Program.

LLM-generated Code Detection and LLM Fingerprint Analysis

Investigating how stylistic and semantic patterns can reveal the authorship of AI-generated source code, aiming to detect LLM-paraphrased code and identify the responsible model through embedding-based and stylometric features.
